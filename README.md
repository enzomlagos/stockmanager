# Stock Manager

Trabajo pre-entrega para curso React JS. La aplicacion esta pensada para ser de utilidad para hacer manejo de stock en un negocio, teniendo un lugar donde visualizar productos y sus detalles asi como manejar stock y registrar compras.

El proyecto todavia esta en estado de crecimiento con algunas necesidades basicas todavia pendientes. Algunas cosas pronto a implementar:
- SearchBar para implementar en la home page, que nos permite buscar productos disponibles
- funcion "compras" que nos permite simular una compra (con un carrito) para hacer baja de stock y registrar informacion importante de la compra como factura, customer info, etc.
- ProductHeadBar componente para agregar una barra superior a los productos con botones que nos permitan: editar la info de un producto, generar duplicados (para luego mapear los modelos de un producto), elegir modelo para ver informacion (dropdown).
- implementacion de base de datos
